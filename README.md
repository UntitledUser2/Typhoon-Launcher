## About

Typhoon Launcher is a community-built tool that aims to make it as easy as possible to get Vortex up and running on Linux. It is not an official product and is developed entirely by the community, for the community.

## Dependencies

The only thing you need is:

- [Bottles](https://usebottles.com/) — used to run Vortex in a Wine environment on Linux

## Important Notice

Typhoon Launcher does not distribute Bottles or the Vortex software. You are responsible for downloading and installing both yourself before using this launcher.

## Download Vortex

You can download Vortex here: [playvortex.io](https://playvortex.io)
